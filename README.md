# Joshua Chen Developer Website

A lightweight static developer portfolio. Open `index.html` directly in a browser or deploy the folder to any static host.

## Edit Projects

Projects are defined in `projects.js`. To add another project, append a new object to the `projects` array with:

- `title`
- `type`
- `image`
- `imageAlt`
- `description`
- `tags`
- `links`

No build step is required.
